# Vrinda Store Annual Sales Dashboard 2022

## Overview
This project is an interactive Excel dashboard created to analyze Vrinda Store's annual sales performance for 2022. The dashboard provides insights into sales trends, customer demographics, order status, sales channels, and regional performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

## Key Metrics Analyzed
- Monthly Orders vs Sales
- Sales by Gender
- Sales by Age Group and Gender
- Order Status Analysis
- Top Performing States
- Sales by Sales Channel

## Dashboard Features
- Interactive filters using Slicers
- Dynamic charts and visualizations
- Customer demographic analysis
- Channel-wise sales comparison
- State-wise sales performance tracking

## Key Insights
- Women contributed a higher share of sales compared to men.
- Most orders were successfully delivered.
- Amazon and Myntra generated a significant share of total orders.
- Maharashtra and Karnataka were among the top-performing states.
- Adult customers contributed the largest share of sales.
